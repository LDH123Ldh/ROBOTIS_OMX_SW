# ROS 2

ROS 2는 로봇 내부의 여러 프로그램을 서로 연결해 주는 통신 시스템입니다.  
로봇 안에는 카메라, 모터, 센서, 자율주행 알고리즘처럼 다양한 기능이 따로 존재합니다.  
ROS 2는 이 기능들이 서로 데이터를 주고받으며 하나의 로봇처럼 작동하도록 연결해 줍니다.

## 왜 ROS 2를 사용하는가?

카메라 프로그램, 모터 제어 프로그램, 자율주행 프로그램을 각각 따로 만든 뒤,  
ROS 2를 이용해 서로 연결하면 전체 로봇 시스템을 더 체계적으로 구성할 수 있습니다.

즉, 모든 기능을 하나의 큰 프로그램에 몰아넣는 것이 아니라,  
기능별로 나누어 개발하고 필요한 데이터만 주고받게 만드는 방식입니다.

## 핵심 개념

### Node

노드는 기능별로 나누어진 독립적인 프로그램입니다.

예를 들어 다음과 같은 노드가 있을 수 있습니다.

- 카메라 영상을 처리하는 노드
- 바퀴를 제어하는 노드
- 장애물을 인식하는 노드
- 자율주행 경로를 계산하는 노드

### Topic

토픽은 노드들이 데이터를 주고받는 일방향 통로입니다.

예를 들어 카메라 노드가 현재 화면 데이터를 계속 보내면,  
객체 인식 노드는 그 토픽을 받아 화면 속 물체를 분석할 수 있습니다.

### Service / Action

서비스와 액션은 명령을 주고받을 때 사용됩니다.

예를 들어 “저 위치로 이동해라”라는 명령을 보내면,  
로봇은 “알겠다”, “이동 중이다”, “도착했다”와 같은 응답을 보낼 수 있습니다.

## 정리

ROS 2는 로봇의 여러 기능을 노드로 나누고,  
토픽·서비스·액션을 통해 서로 연결하는 로봇 소프트웨어 통신 구조입니다.
